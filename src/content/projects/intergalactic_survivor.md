## Description
> A virtual Reality game developed in Unity for a university project within a tight three week time period. The experience features an immersive zero gravity swimming navigation system, a cable routing minigame, a diegetic HUD, a realistic screwdriving mechanic and an inventory system integrated into the player's arms.

## Tech Specs
- Language: **C#**
- Engine: **Unity**

## Key Features & Architecture
- **Zero-Gravity Locomotion**: Developed a physics based VR movement system that requires players to "swim" through the environment using their hands and controllers, closely mimicking the feeling of zero gravity in space.
- **Interactive VR Mechanics**: Developed custom interactive systems including a cable routing minigame and a realistic screwdriving mechanic, leveraging Unity's XR Interaction Toolkit to handle grabbing, twisting and snapping interactions.
- **Diegetic UI & Inventory**: Designed a physical HUD and an arm mounted deposit system, avoiding 2D screen space menus to improve VR immersion and player presence.

## What I learned or improved
- Enhanced proficiency in Unity VR development and the XR Interaction Toolkit, particularly in handling physics based hand interactions and custom player locomotion.
- Improved ability to design and implement diegetic user interfaces that feel natural and intuitive in a virtual reality space.
